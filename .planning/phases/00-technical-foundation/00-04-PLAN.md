  <action>
This task has multiple sequential human steps with potential wait times. The Apple Developer enrollment (D-03) may take up to 48 hours to process.

**Phase A: Prerequisites (do these first, they take time)**

1. **Enroll in Apple Developer Program**
   - Go to https://developer.apple.com/programs/enroll/
   - Sign in with Apple ID, complete enrollment ($99/year)
   - Enrollment can take up to 48 hours to process
   - After approval, note your **Team ID** (10-character code) from developer.apple.com > Membership

2. **Install Xcode from Mac App Store**
   - Search "Xcode" in Mac App Store, install (large download, ~12GB)
   - After install, open Xcode once to accept license agreement and install additional components
   - Verify: `/Applications/Xcode.app` exists
   - Run `sudo xcode-select -s /Applications/Xcode.app/Contents/Developer` to point xcode-select at the full IDE

**Phase B: Godot iOS Export (after enrollment is approved)**

3. **Download iOS export templates in Godot**
   - Godot Editor > Editor menu > Manage Export Templates > Download
   - Wait for download to complete

4. **Create iOS export preset**
   - Project > Export > Add > iOS
   - Set Bundle Identifier: decide on format (e.g., `com.whitedoeinn.lanternkeeper` or personal domain)
   - Set Team ID: paste the 10-character code from step 1
   - Set App Name: "Lanternkeeper" (no spaces -- per Pitfall 6)
   - Leave other settings as default for now

5. **Export to Xcode project**
   - In Export dialog, click "Export Project"
   - Choose an output directory (e.g., `~/Desktop/lanternkeeper-ios/`)
   - Do NOT check "Export with Debug" for the final verification (debug is fine for initial testing)
   - Godot generates a `.xcodeproj`

6. **Build and run in Xcode**
   - Open the generated `.xcodeproj` in Xcode
   - Select your physical iOS device as the build target (connect via USB or WiFi)
   - In project settings: set Minimum Deployment Target to `16.0` (per D-06 -- must set this manually after every export)
   - Configure signing: select your Apple Developer team
   - Build and Run (Cmd+R)
   - Device should show the "Lanternkeeper" label on colored background

7. **Verify on device**
   - App launches without crashing
   - "Lanternkeeper" text is visible
   - Portrait orientation is locked
   - Safe area margins are applied (text not behind notch/Dynamic Island)
   - Check Xcode console for autoload verification prints
